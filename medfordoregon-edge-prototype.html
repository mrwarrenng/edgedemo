WHAT'S ADDED (6 TOUCHPOINTS)
  • Business Mega Menu: 3rd column added alongside existing 2 columns. Existing Dev Services + Bid/Vendor links untouched. EDGE links are additive.
  • Hero Card: Right side of existing OC Rotating Banner. OC Featured Content row.
  • Sidebar Widget: Below "Stay Connected" in right column. WYSIWYG + custom CSS class.
  • Services Row Card: 6th card added to existing services grid. Points to edge.medfordoregon.gov.
  • EDGE Promo Band: New full-width row between Services and Subscribe. OC Featured Content + custom CSS/JS.
  • Footer Column: 4th column added to existing 3-column footer.
WHAT'S NOT CHANGED
  • Business nav link still goes to /Business — all existing secondary & tertiary pages preserved
  • Do It Online tabs: same 3 tabs, same buttons, same links
  • News, Public Notices, Meetings/Calendar: all present with correct links
  • No existing links removed or redirected
  • No departments lose their pages
THE PITCH TO CM & COMMS

"We're not asking for a redesign. We're asking to add 6 elements to the existing homepage — all within OpenCities' documented component library. The Business mega menu keeps every existing link. Development Services, Bid Opportunities, Vendor Registration — all untouched. We're adding a third column to the mega menu, a card in the hero, a widget in the sidebar, one more service card, one promotional section, and a footer column. Every EDGE link routes to edge.medfordoregon.gov, which Econ Dev maintains independently. Comms retains 100% control of medfordoregon.gov."
